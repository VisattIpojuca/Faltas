# Painel de Faltas - APS Ipojuca

Este projeto exibe, em tempo real, as faltas dos profissionais da Atenção Primária à Saúde do município de Ipojuca, com base em dados preenchidos via Google Forms e armazenados no Google Sheets.

## Como usar

### 1. Pré-requisitos

- Conta no GitHub
- Conta no [Streamlit Cloud](https://share.streamlit.io/)

### 2. Subir os arquivos no GitHub

- Faça upload dos arquivos `faltas.py` e `requisitos.txt` em um novo repositório.

### 3. Publicar no Streamlit Cloud

- Conecte sua conta do GitHub
- Escolha o repositório e selecione o arquivo `faltas.py`
- O painel estará online com acesso aos dados da planilha conectada.
